Este proyecto contiene implementaciones en Python de los siguientes métodos numéricos:

- Método de Bisección
- Método de Newton-Raphson
- Método del Trapecio
- Método de Riemann

Incluye:
- Un menú interactivo (`main.py`) para que el usuario seleccione el método a ejecutar.
- Tests unitarios para validar el funcionamiento de cada método.

---

## Requisitos

- Python 3.11 o superior instalado
- Recomendado: Editor como VS Code
- Librería `unittest` (viene incluida con Python)

---

##  Ejecutar el programa con menú

1. Abre la terminal en la carpeta `calculo-numerico`.
2. Ejecuta:

```bash
python main.py

---

##  Ejecutar los test
